# Changelog

## Version 1.0.0 - Initial Release

### ✨ Features
- **Drzewo Zdzisia Structure** - Rare world generation structure
- **Giant Procedural Tree** - 25-40 blocks tall with thick trunk
- **House on Branch** - Small wooden house attached to tree branch
- **Custom Blocks** - Zdzisia Log and Zdzisia Leaves
- **Custom Items** - Zdzisia Acorn (throwable) and Zdzisia Sapling
- **Loot Chests** - Contains Zdzisia items and rare loot

### 🔧 Technical
- Fabric Mod API 1.21.1
- Java 21 compatible
- Procedural tree generation
- Biome integration for Overworld

### 🎨 Content
- 2 custom blocks with textures
- 2 custom items with models
- Fully translated (English)
- Block and item models in JSON

## Installation
1. Install Fabric Loader for 1.21.1
2. Install Fabric API
3. Drop JAR into .minecraft/mods/
4. Launch and enjoy!
