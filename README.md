# Zdzisia Tree Mod

A Fabric mod for Minecraft 1.21.1 that adds **Drzewo Zdzisia** - a rare world structure featuring a giant tree with a house built on a branch!

## Features

🌲 **Giant Procedural Tree**
- Height: 25-40 blocks
- Thick 3x3 trunk
- Natural branches extending from the trunk
- Leafy crown at the top

🏠 **House on a Branch**
- Small wooden house attached to a branch
- Contains 2 loot chests with Zdzisia items
- Crafting table and bed for comfort
- Generates at different heights randomly

🎁 **Custom Items & Blocks**
- **Zdzisia Log** - Custom wood blocks with oak texture
- **Zdzisia Leaves** - Custom foliage blocks
- **Zdzisia Acorn** - Throwable item that can be planted
- **Zdzisia Sapling** - Grows into new Zdzisia trees

## Installation

1. Download the mod JAR file
2. Place it in your `.minecraft/mods/` folder
3. Make sure you have Fabric Loader and Fabric API installed for 1.21.1
4. Launch Minecraft with the Fabric profile

## Building from Source

```bash
./gradlew build
```

The built JAR will be in `build/libs/`.

## Loot Chests

The house contains chests with:
- Zdzisia Acorn (5-8x)
- Zdzisia Sapling (2-3x)
- Zdzisia Log (4-6x)
- Emeralds (1-2x)
- Diamonds (rare)

## Rarity

The Zdzisia Tree structure is **very rare** and generates only in the Overworld. It will appear randomly in your world.
