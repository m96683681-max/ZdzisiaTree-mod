# Zdzisia Tree Mod

[![Minecraft Version](https://img.shields.io/badge/Minecraft-1.21.1-green)](https://minecraft.net)
[![Fabric API](https://img.shields.io/badge/Fabric-API-blue)](https://fabricmc.net)
[![License](https://img.shields.io/badge/License-MIT-yellow)](LICENSE)

A Fabric mod for Minecraft 1.21.1 that adds **Drzewo Zdzisia** - a rare and magical world structure featuring a colossal tree with a house nestled on one of its branches!

## 🌲 Features

### Giant Procedural Tree
- **Height**: 25-40 blocks
- **Trunk**: Thick 3x3 custom wood blocks
- **Branches**: Multiple natural branches extending from the trunk
- **Crown**: Large leafy canopy at the top
- **Custom Wood**: Unique Zdzisia Log and Leaves blocks

### 🏠 House on a Branch
- Small but detailed wooden house
- Built directly on a branch extending from the trunk
- Multiple rooms with furniture
- Loot chests with valuable items
- Crafting table and bed included

### 🎁 Custom Items & Blocks

| Item | Description |
|------|-------------|
| **Zdzisia Log** | Custom log block, used in tree generation |
| **Zdzisia Leaves** | Custom foliage block with drop mechanics |
| **Zdzisia Acorn** | Throwable item, right-click to convert to sapling |
| **Zdzisia Sapling** | Can be planted to grow new trees |

### 📦 Loot Chests
The house contains loot chests with:
- Zdzisia Acorns (5-8x)
- Zdzisia Saplings (2-3x)
- Zdzisia Logs (4-6x)
- Emeralds (1-2x) - rare
- Diamonds - very rare

## 📥 Installation

### Requirements
- Minecraft 1.21.1
- Fabric Loader ([Download](https://fabricmc.net/use/installer/))
- Fabric API ([Download](https://modrinth.com/mod/fabric-api))

### Steps
1. Download the mod JAR from [Releases](https://github.com/m96683681-max/ZdzisiaTree-Mod/releases)
2. Place it in your `.minecraft/mods/` folder
3. Launch Minecraft with Fabric profile
4. Enjoy exploring!

## 🔨 Building from Source

```bash
# Clone repository
git clone https://github.com/m96683681-max/ZdzisiaTree-Mod
cd ZdzisiaTree-Mod

# Build the mod
./gradlew build

# JAR output: build/libs/zdzisia_tree-1.0.0.jar
```

## 🌍 World Generation

- **Rarity**: Very rare - approximately 1 in 10,000 chunks
- **Biome**: Generates in all Overworld biomes
- **Height**: Generates on surface at appropriate Y-level
- **Frequency**: Only one structure per chunk max

### Finding a Tree
- Explore a fresh world
- Use commands: `/locate structure zdzisia_tree:zdzisia_tree`
- Or use mods like Xaero's World Map for easier navigation
